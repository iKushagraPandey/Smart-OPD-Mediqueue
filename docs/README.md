# MEDIQUEUE

MEDIQUEUE is a hospital workflow platform with patient registration, doctor queue management, pharmacy token access, and manager analytics.

## Demo Credentials

- Manager: `admin@manager.in` / `12345678`
- Doctors: `doc1@<department>.in` / `12345678`
- Pharmacy OTP: `123456`

## Modules

- Patient Portal: voice/manual registration, AI symptom mapping, token + QR generation.
- Queue Status: token-based live queue status check.
- Doctor Dashboard: department queue, call next, consultation completion, prescriptions.
- Pharmacy Portal: token verification, prescription fetch, medicine ordering.
- Manager Dashboard: live analytics and department charting.
